# Vortex Core 视觉方向

## 方向

“把一场恒星风暴封进实验室玻璃”。视觉主体始终是原作的黑金 FBM 涡旋和高色散玻璃球；交互只改变原 TSL uniform 与玻璃材质，不增加爆炸碎片、二维光斑、屏幕闪白或伪造粒子。

## 原版基线合同

- 源码锚点：`MisterPrada/vortex-glass-sphere@3587870c...`，MIT。
- 保留 513×513 TSL 实例粒子、原 FBM 五层噪声、粒子阈值、Rogaland HDR、PhysicalNodeMaterial、25° 相机和渐变背景。
- `?baseline=1` 必须显示原 Codrops 标题/链接、Tweakpane、TransformControls helper 和 OrbitControls，不加入产品状态机。
- 禁止以 Canvas 2D、视频、贴图序列或另一套 CPU/GPU 粒子系统代替原作。

## 界面系统

产品版只有三层：左上 `VORTEX CORE` 与 `STUDY 07 / TSL GLASS FIELD`；球心 68 px 的细线蓄能环；底部一句“双指按住涡核”隐藏手势提示。提示在首次有效双指按下后淡出，不再跟随 charging/release/recover 切换动词。文字使用系统等宽字体，冷白色，字距 0.14–0.18em。

单指拖动直接旋转原作相机，不显示文字教学。双指从球心起手时才拥有涡核蓄能手势；蓄能环和原作涡旋必须同步收缩，任一手指松开时真实释放。

产品版 loading 不使用原作者的大三角动画：黑色空场中央只显示直径 46–58 px 的双层细线“涡核呼吸环”与一个微光核心，无文字、无 Logo、无平台标记。`?baseline=1` 仍保留原作者三角预加载动画，作为源码复刻证据；两种模式共用同一资源就绪与退场时机。

## 动效与状态

- idle：保持原作运动与自由旋转；只在未发现隐藏手势时显示“双指按住涡核”。
- charging：环由 0 绘制到 100%，涡旋逐步压暗、收紧；不增加状态文字。
- release：环在 180 ms 内扩到 1.9 倍并消失，原粒子阈值打开、速度和发光升至峰值。
- recover：1450 ms 内回到原版参数，不添加装饰性余波。
- loading：涡核细环以两个相反方向慢速旋转，资源就绪后在 1 s 内熄灭；不借用原作三角形作为产品识别。

## 响应式与无障碍

在 390×844 与 320×568 下玻璃球保持居中且占短边约 96%；标题避开 44 px 顶部安全区，双指提示距底部至少 58 px。单指旋转与双指球心手势按起始位置分流；`prefers-reduced-motion` 下把释放/恢复总时长压缩到 520 ms。
