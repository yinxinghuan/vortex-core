## 检查结论

- P0/P1：无。点击前 `window.experience`、性能 profile 与 renderer QA 对象均不存在，证明 Three/WebGPU 未初始化；点击后每种档位只请求一次 Experience。
- 启动成本：构建后的初始 JS 为 3.33 kB（gzip 1.59 kB）；1.24 MB Experience chunk 只在点击后动态请求。抽象封面不再加载 897.62 kB 海报。
- 运行成本：手机 Medium 从 263169 粒子降到 103041（减少 60.8%），Low 降到 50625（减少 80.8%）；主循环最多一个 `renderAsync` 在途，页面隐藏时停止提交。
- 视觉：为避免过度降质，首轮 25921 / 50625 粒子方案已废弃，复验后将最低档提高至 50625、Medium 提高至 103041；涡旋仍保留连续旋臂与玻璃细节。
- 主体：390×844 和 320×568 均保持玻璃球居中，涡旋占画面主导；无设备外框、卡片或大面积 UI 遮挡。
- 动效：释放只改变原 TSL uniforms。初版把 `radius` 拉到 0.18 时出现蓝色块状低 alpha 粒子，已收敛到 0.94 并把色散峰值降至 5.6，复验后不再出现假碎片感。
- 可读性：标题与提示在两种尺寸均清晰；触摸目标为全屏，大于 44×44 px。
- 回归：产品模式隐藏原作者面板；基线模式继续显示原版全部界面和行为。
- Loading 归属：产品模式使用静止暗场，点击后才让奇点呼吸；不显示上游作者三角动画。基线模式仍保留，署名与复刻证据没有被删除。
